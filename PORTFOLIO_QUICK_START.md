# Portfolio Builder - Quick Start Guide

## For Users

### Creating Your First Portfolio

1. **Login to Dashboard**
   - Navigate to `/dashboard`
   - Click "Create Portfolio" button

2. **Set Your Subdomain**
   - Enter a unique subdomain (e.g., `john-doe`)
   - This will be your portfolio URL: `john-doe.resumeai.com`

3. **Generate Content with AI** (Optional)
   - Click "AI Generate" button
   - AI will create:
     - Hero section with title and description
     - Professional about section
     - Organized skills by category
     - 3-4 project ideas
   - Edit the generated content as needed

4. **Fill in Your Information**
   - **Hero Section**: Your name, title, and introduction
   - **About Me**: Your story and background
   - **Skills**: Add skill categories (Frontend, Backend, etc.)
   - **Projects**: Add your projects with:
     - Name and description
     - Technologies used
     - Upload project images
     - Live demo and GitHub links
   - **Contact**: Email, phone, LinkedIn, GitHub
